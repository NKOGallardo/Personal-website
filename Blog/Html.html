By the end of this post, you'll understand what HTML is, how it works, and you'll have created your very first webpage. Don't worry if you've never written a line of code before â€“ we'll start from the absolute basics and build up step by step.

What is HTML?

HTML stands for HyperText Markup Language. Let's break that down:

  • HyperText: Text that contains links to other text
  • Markup: A system for annotating text to indicate formatting
  • Language: A set of rules and syntax that computers understand

Think of HTML as the skeleton of a website. Just like your body has a skeleton that gives it structure, HTML provides the structure for web pages. It tells the browser what content to display and how it should be organized.

Fun Fact

HTML was created by Tim Berners-Lee in 1991. The first website ever created is still online at info.cern.ch!

The Basic HTML Structure

Every HTML document follows the same basic structure. Here's what a simple HTML page looks like:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

Let's understand each part:

  • <!DOCTYPE html>: Tells the browser this is an HTML5 document
  • <html>: The root element that contains all other elements
  • <head>: Contains metadata about the document (not visible on the page)
  • <title>: Sets the page title (appears in browser tab)
  • <body>: Contains all the visible content of the page

Important

HTML uses "tags" enclosed in angle brackets <>. Most tags come in pairs: an opening tag and a closing tag with a forward slash </>.

Common HTML Tags You Need to Know

Headings

HTML has six levels of headings, from <h1> (largest) to <h6> (smallest):

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>
<h4>Minor Heading</h4>
<h5>Small Heading</h5>
<h6>Tiny Heading</h6>

Text Content

For regular text content, we use paragraphs and text formatting:

<p>This is a paragraph of text.</p>
<p>This is <strong>bold text</strong> and this is <em>italic text</em>.</p>
<p>You can also <u>underline</u> text or <mark>highlight</mark> it.</p>

Lists

HTML supports both ordered (numbered) and unordered (bullet) lists:

<!-- Unordered List -->
<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

<!-- Ordered List -->
<ol>
    <li>Step one</li>
    <li>Step two</li>
    <li>Step three</li>
</ol>

Links and Images

Make your pages interactive with links and visual with images:

<!-- Links -->
<a href="https://www.google.com">Visit Google</a>
<a href="mailto:someone@example.com">Send Email</a>

<!-- Images -->
<img src="image.jpg" alt="Description of the image">

Accessibility Tip

Always include an alt attribute for images. It helps screen readers describe images to visually impaired users.

Let's Build Your First Webpage!

Now it's time to put everything together. Here's a complete example of a simple personal webpage:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>About Me - My Personal Page</title>
